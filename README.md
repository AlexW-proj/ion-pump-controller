# ion-pump-controller
open source controller that takes 24V in and spits out 0 - 7000V, set by a knob with an adjustable ramp time. still just theoretical design, not tested physically

IMPORTANT: in the schematic, there is no output! Ion pumps may have a few different input types and I dont know what they are, so you must insert your own! in the bottom left of the schematic, look for "final Vout", and put your desired output connector across it!
